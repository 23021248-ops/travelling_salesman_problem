# Traveling Salesman Problem Solver

## Introduction

This project solves the Traveling Salesman Problem (TSP) using:

- A* Search
- Branch and Bound
- Heuristic Functions

## Problem Description

Given a set of cities and distances between them, find the shortest Hamiltonian cycle that visits each city exactly once and returns to the starting city.

## Run

```bash
python TSP-Heuristic-Search.py
```
